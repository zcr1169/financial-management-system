## 技术栈

- **语言**: Java 21
- **GUI**: Swing + FlatLaf 3.5.4
- **数据存储**: Excel (.xlsx) via Apache POI 5.3.0
- **构建**: Gradle

## 功能模块

- 📋 会计科目管理（4-2-2-2 编码体系，多级科目自动汇总）
- 📝 凭证填制与审核（三岗分离：制单人≠审核人≠记账人）
- ✍️ 出纳签字
- 📒 记账与结账
- 📊 报表查询（科目余额表、试算平衡）
- 👤 用户与权限管理
- 📤 导出 PDF / Excel
- 📖 操作日志
