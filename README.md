# 财务信息管理系统

北京林业大学 · 2025-2026学年第二学期  
**《财务信息管理》课程大作业**

---

## 技术栈

- **语言**: Java 21
- **GUI**: Swing + FlatLaf 3.5.4
- **数据存储**: Excel (.xlsx) via Apache POI 5.3.0
- **构建**: Gradle

## 功能模块

- 📋 会计科目管理（4-2-2-2 编码体系，多级科目自动汇总）
- 📝 凭证填制与审核（三岗分离：制单人≠审核人≠记账人）
- ✍️ 出纳签字
- 📒 记账与结账
- 📊 报表查询（科目余额表、试算平衡）
- 👤 用户与权限管理
- 📤 导出 PDF / Excel
- 📖 操作日志

## 快速开始

### 运行 EXE

进入 `EXE文件/` 目录，双击运行 `财务管理系统.exe`

### 从源码运行

```bash
cd 源代码
gradlew run
```

## 项目结构

```
├── 源代码/              # Java 源码（Gradle 项目）
├── EXE文件/             # jpackage 打包的 Windows 可执行文件
├── 财务管理系统实验报告.docx  # 课程实验报告
└── DATA.xlsx            # 数据文件（不包含在仓库中）
```
